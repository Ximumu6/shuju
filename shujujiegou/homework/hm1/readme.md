第二章向量的代码作业
1. 以书p30的代码2.1为头文件（Vector.h），实现Vecter模版类（Vector.cpp）
2. 定义一个复数类，随机生成一个无序的复数向量（有重复项）
（1）在主函数中测试无序向量的置乱、查找（实部和虚部均相同）、插入、删除和唯一化的操作
（2）以复数的模为基准进行排序（模相同的情况下，以实部为基准），比较顺序、乱序、逆序的情况下，起泡排序和归并排序的效率（clock()函数记时）
（3）在顺序的复数向量中，实现区间查找算法，查找出模介于[m1,m2) 的所有元素，按序存于一个子向量中作为返回值。

实验目的：根据书本上向量的头文件，编译一个能对复数数组进行置乱、查找（实部和虚部均相同）、插入、删除、区间查找、排序和唯一化的操作
代码概要：
 1、通过randomize函数对向量数组进行随机赋值
 2、通过find函数进行查找：成功输出下标，失败输出查找失败
 3、通过insert函数插入元素
 4、通过remove函数删除元素
 5、利用起泡排序和归并排序对数组进行顺序、逆序、乱序的排序
 6、利用unsort函数进行去重
 7、利用abssort函数进行区间查找

遇到困难：
 1、由于采用编译软件是dvc++11版本，版本太低，而auto 返回类型“没有尾随返回类型”是 C++14 的一个特性，所以得在后面加一句 -> std::size_t
 2、Dev-C++：#error This file requires compiler and library support for the ISO C++ 2011 解决方案：先在dev的【工具】里找到【编译选项】，在这个【编译时加入以下命令】处打钩,然后在空白栏输入【-std=c++11】,确定.然后就能支持c++11
